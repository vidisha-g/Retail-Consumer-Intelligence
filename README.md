# Retail Consumer Intelligence Dashboard

An end-to-end Data Analytics project that analyzes retail consumer purchasing behavior using Python, SQL, Excel, and Power BI to generate actionable business insights.

---

# Project Overview

Retail Consumer Intelligence is an end-to-end Data Analytics project that analyzes retail customer purchasing behavior to uncover valuable business insights. The project demonstrates the complete analytics workflow, starting from raw data preprocessing in Python to SQL analysis, Excel dashboard creation, and an interactive Power BI dashboard.

The objective is to help businesses understand customer behavior, identify sales trends, evaluate product performance, and support data-driven decision-making.

---

# Business Problem

Retail businesses generate a large amount of customer transaction data every day. However, without proper analysis, it is difficult to identify:

- Best-selling product categories
- Customer purchasing behavior
- Sales trends across seasons
- Preferred payment methods
- Loyal customer contribution
- Subscription impact on sales
- Product size preferences

This project converts raw retail data into meaningful business insights using modern data analytics tools.

---

# Project Objectives

- Clean and preprocess raw retail data
- Perform Exploratory Data Analysis (EDA)
- Engineer new features for better analysis
- Analyze data using SQL queries
- Build an interactive Excel dashboard
- Develop an interactive Power BI dashboard
- Generate business insights for business decision-making

---

# Dataset Information

**Dataset:** Retail Consumer Behavior Dataset

**Source:** Kaggle

**Total Customers:** ~3,900

**Total Sales:** 233K+

The dataset contains customer demographics, purchase history, product information, payment methods, shipping details, ratings, subscription status, and other retail attributes.

# Key Business Insights

1. Clothing generated the highest overall sales.
2. Male customers contributed more revenue than female customers.
3. M-size products were purchased the most.
4. Credit Card was the most preferred payment method.
5. Fall season recorded the highest sales.
6. Loyal customers contributed significantly to total revenue.
7. Subscription status influenced purchasing behavior.
8. Different shipping methods generated comparable sales performance.
